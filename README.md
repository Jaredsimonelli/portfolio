# Jared Portfolio

This repository now hosts the production portfolio as a plain static site for GitHub Pages.

## Publishing

GitHub Pages should continue publishing from `main` / `docs`.

The live site contents are in `docs/`, including `docs/CNAME` for the custom domain.

## Local editing

Edit the published files directly:

- `docs/index.html`
- `docs/styles.css`
- `docs/script.js`
- `docs/assets/`
